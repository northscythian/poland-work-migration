## 🚀 Процесс трудоустройства

Трудоустройство в Польше для граждан Казахстана состоит из двух ключевых этапов: **действия со стороны работодателя** (получение разрешения на работу) и **действия с вашей стороны** (оформление национальной визы типа D).

```mermaid
flowchart LR
    A[Поиск работодателя] --> B[Work Permit\nот работодателя]
    B --> C[Сбор документов\nв Казахстане]
    C --> D[Подача на визу D\nв Консульство/VFS]
    D --> E[Переезд и\nКарта Побыту]
