Информационный репозиторий со списком документов и пошаговым процессом официального трудоустройства граждан Республики Казахстан в Республике Польша.

---

## 🗺 Пошаговый алгоритм (Workflow)

```mermaid
graph TD
    A[Поиск работодателя в Польше] --> B[Тест рынка труда работодателем]
    B --> C[Получение разрешения Zezwolenie Typ A via praca.gov.pl]
    C --> D[Пересылка документа в Казахстан]
    D --> E[Сбор документов и запись в e-Konsulat / VFS Global]
    E --> F[Подача на национальную визу типа D]
    F --> G[Переезд и регистрация по месту жительства Zameldowanie]
